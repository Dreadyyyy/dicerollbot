Installation guide

Download repo
```
git clone "link"
```
Change directory
```
cd "name"
```

Create virtual environment
```
python -m venv .venv
```

Activate virtual environment
```
source .venv/bin/activate
```

Install dependencies
```
pip install -r requirements.txt
```

Create .env file
```
echo BOT_TOKEN=your_api_token>.env
```

Run bot
```
python main.py
```
